About Bhojpur Wealth

The Bhojpur Wealth is a comprehensive wealth management application for individuals and enterprises to keep track of stocks, ETFs or cryptocurrencies and make solid, data-driven investment decisions. The source code is fully available as an open source software. The project has been initiated by Bhojpur Consulting Private Limited, India and is driven by the efforts of its contributors. This instance is running Bhojpur Wealth {{ version }}. Check the system status at status.bhojpur-consulting.com.

If you encounter a bug or would like to suggest an improvement or a new feature, please join the Bhojpur Wealth Slack community, tweet to @bhojpurconsult, send an e-mail to support@bhojpur-consulting.com or open an issue at GitHub.
